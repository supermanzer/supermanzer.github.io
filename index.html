Data Processing

In both my work building business administration applications and my academic studies in oceanography, figuring out efficient and clever ways to find, extract, and process data has been a critical component. I even find such activities enjoyable. Here I will mostly be focusing on how I can turn a simple static page into a Javascript powered data visualization platform.

Data Visualization

Much of the work I did for my MS invovled visualizing oceanographic data using Matlab. While it's an extremely useful platform, I want to expand the reach of my data visualizations and not be tethered to a license. Thus I am turning to Javascript and the D3 plotting library to develop interactive plots of oceanographic data. This is an ongoing pursuit and the graphs I display here will be regularly updated.

Web Development

I think the web is a great vehicle for communicating science that has yet to be fully realized. I would like to explore general web development to better understand how I can use structure, style, and interactive components to communicate and engage others. However I am still quite a novice and will be relying on many more talented developers to aid me in my quest. I have to thank all the many contributors at MaterializeCSS for this lovely template and many spiffy components.

My focus is on the ocean...in case you haven't noticed yet.

Contribute

As I'm a novice at much of this work I would be happily invite any interested parties to contribute. I will be attempting to build out data parsing Javascript components as well as interactive plots of data. In some cases I may have to rely on server side code (Python 3) to handle data wrangling and write out to a static file. In those cases I will host both the static data file and the Python script in this repository for anyone to examine, fork, or comment on.
